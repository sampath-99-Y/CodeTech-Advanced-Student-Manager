# 🎓 Advanced Student Manager (JavaFX)

A modern JavaFX-based application to manage student records with an interactive UI, animations, and data visualization.

## ✨ Features
- Add / Delete Students
- Search Functionality
- Statistics (Average, Highest, Lowest)
- Interactive Bar Chart
- Clean UI with CSS styling
- Smooth animations

## 🛠 Technologies Used
- Java
- JavaFX
- CSS

## ▶️ How to Run

```bash
javac --module-path "PATH_TO_FX/lib" --add-modules javafx.controls Main.java Student.java
java --module-path "PATH_TO_FX/lib" --add-modules javafx.controls Main
